# ProfilPersonnel

## Description

Page statique de présentation personnelle de Patrice NGOULOU-NGOULOU. Le site présente une biographie, les compétences, la formation, les liens sociaux et un formulaire de contact.

## Fichiers

- `profil.html` : page principale.
- `styles.css` : styles du portfolio.
- `images/photo-profil.jpg` : image de profil attendue.

## Fonctionnalités

- Présentation personnelle avec photo.
- Section compétences.
- Détails de formation et domaines d’intérêt.
- Liens vers GitHub et LinkedIn.
- Formulaire de contact statique.

## Utilisation

1. Ouvrir `profil.html` dans un navigateur.
2. Vérifier que `styles.css` est présent.
3. Vérifier que l’image de profil est disponible dans `images/`.

## Notes

Le formulaire de contact est visuel, sans traitement côté serveur.
